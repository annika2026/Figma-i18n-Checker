#### 核心方法：

```javascript
// 精确计算文本宽度
calculateTextWidth(text, fontStyle) {
  // 使用 Canvas measureText API
  // 完全模拟浏览器渲染
  // 返回精确的像素宽度
}

// 严格检查适配（无阈值）
checkTextFit(text, containerWidth, fontStyle) {
  const textWidth = this.calculateTextWidth(text, fontStyle);
  const fits = textWidth <= containerWidth;  // 严格比较
  
  return {
    textWidth,      // 精确宽度
    fits,           // 是否适配
    overflow,       // 超出像素
    overflowPercentage  // 超出百分比
  };
}
```

## 🔍 技术细节

### Canvas 测量原理

```javascript
// 1. 创建 Canvas 上下文
const canvas = createCanvas(1, 1);
const ctx = canvas.getContext('2d');

// 2. 设置完整的字体
ctx.font = 'italic 500 16px Inter, sans-serif';

// 3. 精确测量
const metrics = ctx.measureText(text);
let width = metrics.width;

// 4. 应用字间距
if (letterSpacing) {
  width += letterSpacing * (text.length - 1);
}
```

### 支持的字体

内置 fallback 映射：
- Inter, SF Pro Display, Roboto
- PingFang SC, Microsoft YaHei (中文)
- Helvetica, Arial
- Noto Sans, Open Sans
